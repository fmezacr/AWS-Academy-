# AWS Academy TEC

Repositorio con notebooks utilizadas en los cursos de **AWS Academy impartidos en el Tecnológico de Costa Rica (TEC)**.

Cada notebook incluye explicaciones claras, comentarios en el código y descripciones de los resultados para facilitar su comprensión, especialmente para personas con poca experiencia en programación o en servicios de AWS.

## Contenido

El repositorio incluye materiales de cursos relacionados con:

* Fundamentos de computación en la nube
* Aprendizaje automático
* Procesamiento de lenguaje natural
* Inteligencia artificial generativa
* Amazon SageMaker
* Amazon Comprehend
* Amazon BlazingText
* XGBoost y evaluación de modelos

## Organización

Las notebooks siguen el flujo original de los laboratorios y mantienen sin cambios:

* El código
* Las variables
* Las funciones
* El orden de ejecución
* Las gráficas
* Los resultados esperados

La documentación adicional explica:

* Qué hace cada bloque de código
* Para qué se utiliza cada biblioteca
* Qué se realiza en cada etapa
* Cómo interpretar los resultados principales

## Uso

1. Ingrese a la carpeta del curso correspondiente.
2. Abra la notebook.
3. Ejecute las celdas en el orden establecido.
4. Consulte los comentarios y explicaciones incorporados.

Algunos laboratorios requieren ejecutarse dentro del entorno de **AWS Academy** debido al uso de permisos, recursos y servicios específicos de AWS.

## Consideraciones

Este repositorio es un recurso de apoyo académico. No sustituye las instrucciones oficiales de los laboratorios ni la documentación de AWS.

No publique credenciales, claves de acceso, tokens ni información sensible dentro de las notebooks.
